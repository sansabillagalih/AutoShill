# Bot Auto Shill Telegram


# Usage:

    1. Download script

    2. Install Python 3+, ketika di tampilan awal install, liat di ujung kiri bawah, ceklis Add to Path. 

    3. Buka cmd ketik: pip install telethon, lalu enter
    
    4. Run script nya
